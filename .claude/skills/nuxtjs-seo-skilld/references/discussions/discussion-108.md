# Nuxt SEO beta feedback

Nuxt SEO Kit v2 (renamed to Nuxt SEO) is in beta, providing:

- Hundreds of bug fixes that can't be ported to v1
- Runtime support (no prerendering required)
- Heaps of new features

See the Installation guide, Migration guide or jump in the <a href="https://discord.gg/275MBUBvgP">Discord</a> for support.

This discussion is for gathering feedback that will help shape the future of the SEO for Nuxt.

I need help with feedback on making sure the stable release of Nuxt SEO is a great DX. Please help me out with feedback on anything, some suggestions:
- Documentation that needs more work
- Undocumented features / breaking changes
- Missing features
- etc
