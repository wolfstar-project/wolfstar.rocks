# Typography

The WolfStar typography system uses Whitney for sans-serif text and the system monospace stack for code.

Typography values are defined in `app/assets/css/utilities.css` and configured via `@nuxt/fonts` in `nuxt.config.ts`.

## Font Sizes

The typography scale follows the standard Tailwind/Wind4 preset:

- `xs` = 0.75rem (12px)
- `sm` = 0.875rem (14px)
- `base` = 1rem (16px)
- `lg` = 1.125rem (18px)
- `xl` = 1.25rem (20px)
- `2xl` = 1.5rem (24px)
- `3xl` = 1.875rem (30px)
- `4xl` = 2.25rem (36px)
- `5xl` = 3rem (48px)

## Line Height

Base line height is set to **1.6** for optimal readability. This provides comfortable spacing for multi-line text blocks while maintaining density for UI components.

## Typography Features

- **Font smoothing**: Antialiased rendering (`-webkit-font-smoothing: antialiased`) applied globally
- **Text rendering**: `optimizeLegibility` applied globally for consistent cross-browser rendering
