# Colors

The WolfStar color system uses Nuxt UI semantic tokens with support for light, dark, and midnight themes.

Colors are defined in `app/assets/css/main.css` using `@theme` scales and `--ui-*` CSS variables. Values shown reflect the **dark** theme (default).
