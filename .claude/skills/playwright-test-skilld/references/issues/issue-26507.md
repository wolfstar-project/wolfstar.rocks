Created a feature request to collect upvotes and see if anyone else would find it useful. Reference - https://github.com/microsoft/playwright/issues/26498

In my case, it would be useful if I could deploy Playwright workers with some balancer in Kubernetes cluster. That way, I could have a lightweight CI job (CI has limited resources anyway) and use more resourceful k8s cluster to deploy Playwright workers and run tests on them.
