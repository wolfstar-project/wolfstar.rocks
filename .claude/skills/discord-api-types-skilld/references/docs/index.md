### About

discord-api-types is a simple module that helps you work with the data returned from Discord's API.With this module, you're able to work with raw data while also making sure what you access exists, and what type it may be.

### Key Features

- Supports both Node.js and Deno
- Supports both CommonJS and ESM
- Receives updates as fast as they are documented
